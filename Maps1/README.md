# **¿Qué rutas toman los alimentos que llegan a Antioquia?**

Muestra las importancias de cada una de las rutas viales de abastecimiento del país, ayuda a ver desde que zonas del país provienen los productos alimenticios. Además de ver las vias también se muestra la representación porcentual de alimentos que transita por cada vía. Este cuenta con información a nivel de producto, mes y año.
